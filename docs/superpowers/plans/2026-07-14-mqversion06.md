# MQNet v0.6.x Upgrade Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Upgrade MQNet to track the mq upstream v0.6.5 release and expose the new FFI surface area (engine config API, `mq_version`, optimization levels, module loading, variable injection, and HTTP cache control) via idiomatic C# wrappers.

**Architecture:** The native `mq_ffi` library already ships all the new functions in v0.6.x; this plan is purely a managed-code wrapper effort. New capabilities are added to `MqEngine` (stateful, multi-call engine) and/or `MqQueryBuilder` (fluent single-call path). New value types / enums are kept in their own files. All changes land on branch `mqversion06`.

**Tech Stack:** C# 13 / .NET 8 + 10, xUnit, P/Invoke (`LibraryImport`), PowerShell (build script), GitHub Actions.

## Pre-flight: What changed in mq-ffi v0.6.x (relevant to this wrapper)

The following new public symbols are exported by `mq_ffi` from v0.6.2 onwards (confirmed via `crates/mq-ffi/src/lib.rs` at tag `v0.6.5`):

| C symbol | Purpose |
|---|---|
| `mq_version()` | Returns a static `const char*` (do NOT free). |
| `mq_set_optimization_level(engine, level)` | Sets AST optimizer level (None/Basic/Full). |
| `mq_set_max_call_stack_depth(engine, u32)` | Caps recursion depth. |
| `mq_set_search_paths(engine, paths, len)` | Module search paths. |
| `mq_define_string_value(engine, name, value)` | Injects a named string constant. |
| `mq_import_module(engine, name)` | Namespace-imports a module (returns error or null). |
| `mq_load_module(engine, name)` | Loads a module into calling scope (returns error or null). |
| `mq_set_http_allowed_domains(engine, domains, len)` | HTTP import allowlist. |
| `mq_clear_http_cache(engine)` | Clears cached HTTP modules (returns error or null). |
| `mq_clear_http_cache_all(engine)` | Full HTTP cache clear (returns error or null). |

The `MqOptimizationLevel` enum in Rust maps to C integers: `None=0`, `Basic=1`, `Full=2`.

The linting feature (`mq-lint` crate) is **not exposed via mq-ffi**; there is no C ABI for it. We will **not** wrap lint in this plan — it is a CLI-only feature.

## Task 5: Expose new capabilities on `MqEngine`

**Files:**
- Modify: `src/MQNet/MqEngine.cs`

This task adds a `Version` static property, and 8 instance methods. The tricky parts are:
- `mq_set_search_paths` / `mq_set_http_allowed_domains` take `char**` (array of C string pointers). We must pin each string as a `byte[]` and pass a pointer array.
- `mq_import_module` / `mq_load_module` / `mq_clear_http_cache` / `mq_clear_http_cache_all` return a nullable error string that must be freed.
- `mq_version` returns a **static** pointer — **do not call `MqFreeString` on it**.

### Helper: marshalling a string array to `char**`

Because `DisableRuntimeMarshalling` is active, we cannot use `[MarshalAs(UnmanagedType.LPArray)]`. We use a helper that allocates a contiguous array of `IntPtr`, pins each UTF-8-encoded string, and passes the array pointer.

Add these private helpers inside the `MqEngine` class (before or after `MarshalResult`):

```csharp
/// <summary>
/// If <paramref name="errorPtr"/> is non-zero, reads the error message, frees the native string,
/// and throws <see cref="MqException"/>.
/// </summary>
private static void ThrowIfError(IntPtr errorPtr)
{
    if (errorPtr == IntPtr.Zero) return;
    var message = Marshal.PtrToStringUTF8(errorPtr) ?? "Unknown error";
    NativeMethods.MqFreeString(errorPtr);
    throw new MqException(message);
}

/// <summary>
/// Marshals a managed string list to a native <c>char**</c> and invokes <paramref name="action"/>.
/// Pinned memory is released before this method returns.
/// </summary>
private static unsafe void MarshalStringArray(IReadOnlyList<string> items, Action<IntPtr, int> action)
{
    int count = items.Count;
    if (count == 0)
    {
        action(IntPtr.Zero, 0);
        return;
    }

    // Encode each string as a null-terminated UTF-8 byte array
    var encodings = new byte[count][];
    for (int i = 0; i < count; i++)
        encodings[i] = System.Text.Encoding.UTF8.GetBytes(items[i] + '\0');

    // Pin each encoding and build the pointer array
    var handles = new System.Runtime.InteropServices.GCHandle[count];
    var ptrs    = new IntPtr[count];
    try
    {
        for (int i = 0; i < count; i++)
        {
            handles[i] = System.Runtime.InteropServices.GCHandle.Alloc(encodings[i], System.Runtime.InteropServices.GCHandleType.Pinned);
            ptrs[i]    = handles[i].AddrOfPinnedObject();
        }
        fixed (IntPtr* ptrArray = ptrs)
        {
            action((IntPtr)ptrArray, count);
        }
    }
    finally
    {
        foreach (var h in handles)
            if (h.IsAllocated) h.Free();
    }
}
```
